# Game-N-Tech

Voici mon premier projet personnel en tant que développeur web, dédié aux addicts du gaming.  

Le but de ce site est de rediriger les utilisateurs vers des pionniers du gaming, en leurs proposants les meilleurs sites de jeux-vidéos et profiter des meilleurs produits à un rapport qualité prix quasi imbattable sur le marché.

Le site a été fait entièrement grâce à React, en utilisant du Javascript pour les fonctionnalitées et tailwindCss pour le visuel. 

Ps: le site est actuellement en cours de développement mais reste entièrement accessible pour une démo.
